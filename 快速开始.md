# YiShe Extensions 快速开始

## 1. 安装依赖

```bash
npm install
```

## 2. 本地开发

```bash
npm run dev
```

## 3. 生产构建

```bash
npm run build
```

构建完成后，浏览器扩展目录为：

```text
.output/chrome-mv3/
```

## 4. 浏览器加载

1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择 `.output/chrome-mv3/`

## 5. 主要页面

- `popup.html`：插件弹窗
- `login.html`：登录页
- `control.html`：控制台页

`login.html` 和 `control.html` 由 WXT 作为 unlisted page 输出，可通过扩展内部跳转打开。

## 6. 目录说明

- `src/entrypoints/`：WXT 页面入口
- `src/background/`：后台逻辑
- `src/content/`：content script 逻辑
- `src/components/`：Vue 组件
- `src/composables/`：状态与逻辑复用
- `src/shared/`：API、配置与扩展通信共享模块
- `public/assets/`：静态资源

旧版 `manifest.json`、`popup/`、`pages/`、`background/`、`content/`、`utils/` 目录已经移除，后续开发直接在新结构上继续。

## 7. 发布目录

```bash
npm run build:release
```

生成目录：

```text
dist/yishe-extensions-v{version}/
```

这个目录可以直接打包或提供给他人作为已解压扩展加载。
